Teaching Philosophy
원본 그대로.
변형 없이. 타협 없이.

클래식 필라테스는 Joseph H. Pilates가 설계한 정확한 순서와 원칙이 있습니다. 유행에 맞춰 변형된 동작이 아닌, 100년간 검증된 오리지널 리퍼토리만을 지도합니다. 몸은 정직합니다. 올바른 움직임만이 몸을 진짜로 바꿉니다.

Certifications & Qualifications
자격 & 경력
01
M.A.T. Pilates Teacher Certification
Basic / Intermediate / Advance System
미국 최고 권위 인증
미국에서 가장 엄격한 클래식 필라테스 자격. Dorothee Vandewalle가 직접 수여합니다.

M.A.T.(Metropolitan Authentic Training)는 Dorothee Vandewalle가 창시한 클래식 필라테스 티처 트레이닝 프로그램입니다. 미국 내 가장 엄격하고 포괄적인 인증 과정으로, 수료까지 700시간 이상이 소요됩니다. 전 세계 클래식 필라테스 스튜디오와 강사들이 인정하는 최상위 자격 중 하나입니다.

Dorothee는 Romana Kryzanowska에게 직접 사사했으며, 30년 이상 클래식 필라테스를 지도해온 세계 최고의 마스터 티처입니다. 그 문하에서 직접 수료한 강사는 전 세계적으로 극소수입니다.
02
Built Pilates
Basic / Intermediate / Advance System
Jung Ah Teacher Certification
Dorothee 정규 과정 이수
Dorothee의 클래식 필라테스 티처 트레이닝 정규 과정을 모두 이수한 자격입니다.

Built Pilates는 Dorothee Vandewalle의 클래식 필라테스 철학과 교수법을 체계화한 프로그램입니다. Basic부터 Advance까지 전 단계를 이수하며, 각 단계에서 까다로운 실기·이론 평가를 통과해야만 자격이 부여됩니다.

03
Built Pilates by Dorothee Vandewalle
Classic Pilates Workshop
마스터와의 직접 수업
Dorothee Vandewalle의 한국 방문 워크샵에 직접 참가했습니다.

Dorothee Vandewalle가 한국을 직접 방문해 진행한 클래식 필라테스 워크샵입니다. 세계 최정상의 마스터로부터 직접 피드백과 교정을 받는 극히 드문 기회로, 이 경험은 클래식 필라테스의 본질을 더욱 깊이 이해하는 계기가 됐습니다.

04
Peak Pilates PPC
Level 1 / 2 / 3
전 세계 35개국 공인 · 최상위 등급
전 세계 35개국이 인정하는 국제 클래식 필라테스 최상위 자격. Level 3는 최고 등급입니다.

Peak Pilates PPC(Pilates Professionals Certification)는 클래식 필라테스의 모든 기구(Mat, Reformer, Cadillac, Chair, Barrel, Ped-o-Pul 등)에 대한 완전한 이해와 지도 능력을 검증하는 국제 자격입니다. Level 3 취득까지 600시간 이상의 수련과 엄격한 실기·이론 평가를 통과해야 합니다.

Level 3는 단순한 동작 지도를 넘어, 체형 분석·움직임 패턴 교정·특수 집단(임산부, 재활 등) 지도 능력까지 포함하는 최상위 레벨입니다.
05
JY Classic Pilates Association
Pre / Post Natal Pilates Program
임신·출산 전후 전문 자격
임신 중과 출산 후, 안전하고 효과적인 필라테스 지도를 위한 전문 자격입니다.

산전·산후 여성의 신체 변화, 금기 동작, 단계별 안전한 움직임 설계 능력을 검증하는 전문 자격입니다. 임신 초·중·후기별 맞춤 지도, 출산 후 골반저근 및 코어 회복 프로그램 설계 능력을 포함합니다.
